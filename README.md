# ATS Scorer Pro 🚀

A premium, purely client-side Applicant Tracking System (ATS) Scorer designed to help job seekers optimize their resumes against specific job descriptions. 

Built with an emphasis on **privacy, speed, and modern UI/UX**, this tool processes everything directly in your browser—no backend servers, no data collection.

## ✨ Features

- **In-Browser PDF Parsing**: Upload your resume as a PDF and watch the offline `pdf.js` engine instantly extract your text. 
- **Smart Keyword Extraction**: Automatically filters out stop-words to find the core technical and soft-skill keywords in any job description.
- **Actionable AI Advice**: Dynamically generates personalized recommendations based on your unique match score and missing keywords.
- **Premium Design System**: Features a custom dark mode, glassmorphism UI elements, gradient typography, and smooth CSS micro-animations.
- **Zero Configuration**: Built with Vanilla HTML/CSS/JS. Deploy it anywhere instantly without `npm install` or build steps.

## 🛠️ Technology Stack
- **HTML5** & **CSS3** (Custom properties, Flexbox/Grid, Animations)
- **Vanilla JavaScript** (ES6+)
- **PDF.js** (Mozilla's open-source library for client-side PDF text extraction)
- **Deployed on Vercel**

## 🚀 How to Run Locally

Because this project is built entirely without heavy frameworks, running it locally is completely effortless:

1. Clone the repository:
   ```bash
   git clone https://github.com/YourUsername/ats-scorer.git
   ```
2. Navigate to the project folder:
   ```bash
   cd ats-scorer
   ```
3. Open `index.html` in any modern web browser. That's it!

## 💡 Why I Built This

I built this tool because I frequently found myself trying to guess whether my resume would pass through automated ATS filters. While there are many tools out there, most require creating an account, paying a subscription, or surrendering your personal resume data to a remote server. 

ATS Scorer Pro was built to solve this: it’s entirely free, instantaneous, and 100% private.
